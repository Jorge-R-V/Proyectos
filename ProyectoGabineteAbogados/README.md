# Landing Page: Gabinete de Abogados (Maziul & Asociados)

Sitio web corporativo y profesional diseñado para el despacho de abogados trilingüe (Alemán, Español e Inglés) **Maziul & Asociados**, con sede física en Valencia, España.

---

## Características del Sitio

- **Diseño Premium y Responsivo**: Adaptado para una óptima lectura tanto en pantallas de escritorio como en dispositivos móviles.
- **Enfoque Internacional**: Soporte visual para el cambio de idiomas (Español, Alemán, Inglés) adaptado a clientes internacionales.
- **Secciones Detalladas**:
  - **Presentación del Abogado Principal**: Perfil dedicado a Klaus Maziul, abogado alemán colegiado en España.
  - **Listado de Servicios**: Información clara sobre las áreas de práctica jurídica.
  - **Miembros del Equipo**: Presentación de los abogados asociados.
  - **Testimonios de Clientes**: Apartado de opiniones reales.
  - **Formulario de Consulta**: Sección interactiva de contacto.
  - **Ubicación Dinámica**: Integración interactiva de mapa a través de Google Maps.
  - **Suscripción a Newsletter**: Pie de página con enlaces legales y redes sociales.

---

## Estructura y Tecnologías

- **HTML5**: Marcado semántico optimizado para accesibilidad y SEO.
- **Sass (SCSS)**: Hojas de estilo estructuradas con variables, anidamiento y modularidad para facilitar el mantenimiento del diseño.
- **CSS3**: Archivo compilado `style.css` listo para producción.

---

## Instrucciones para Desarrollo y Visualización

### 1. Visualización Directa
Puedes visualizar el sitio directamente abriendo el archivo `index.html` en cualquier navegador web.

### 2. Edición de Estilos (Sass)
Si deseas realizar cambios en los estilos visuales, debes editar el archivo `style.scss`. Para compilar estos cambios a CSS, puedes utilizar la extensión **Live Sass Compiler** de VS Code o ejecutar el compilador de Sass por consola:
```bash
sass style.scss style.css --watch
```
Esto mantendrá actualizado el archivo `style.css` de manera automática al guardar cambios en `style.scss`.
